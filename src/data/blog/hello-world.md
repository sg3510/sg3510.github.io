# LLM agents aren't quite ready but workflows are 

This is my first blog post on my newly migrated Astro site.

## Why I moved to Astro

I recently migrated this site from Create React App to Astro. The main reasons were:

- **Better markdown support** - Content lives in `.md` files that are easy to edit
- **Faster builds** - Astro uses Vite under the hood
- **Smaller bundles** - Only ships JavaScript where needed
