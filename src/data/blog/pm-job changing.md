# How to be an effective PM with LLMs

I really like working with AI tools :D

AI is requiring PMs to be more technical again

Getting prototypes together is now easy: Being able to quickly prototype a feature to demonstrate its value is now a very useful skill. before having PMs code would not be a good use of time.

With coding velocity being faster (eg Anthropic shipping full features in just a few weeks) it does not mean that we should be runnign around implmenting all the features. Understand which features to implement is a very powerful PM skill.

PM is still the voice of the product and in charge of evangelizing it
